Getting Started

Quick Start

Include the CSS file and start using classes immediately:

<!DOCTYPE html> <html> <head> <link href="wireframe-ui-lite.css" rel="stylesheet"> </head> <body> <div class="wf-card"> <h1 class="wf-text">Hello World</h1> <button class="wf-btn wf-btn-primary">Get Started</button> </div> </body> </html>

Hello World

Why Wireframe UI Lite?

  • Ultra-lightweight: Only ~5KB gzipped
  • LLM-friendly: Simple, predictable class names
  • No dependencies: Pure CSS, works everywhere
  • Wireframe aesthetic: Perfect for prototyping
  • Mobile responsive: Works on all devices
